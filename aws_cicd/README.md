# Spring Boot HelloWorld Project

This is a simple Spring Boot project with a HelloWorld REST controller.

## Requirements
- Java 17+
- Maven

## How to Run

1. Compile the project:
   ```bash
   mvn clean package
   ```
2. Run the application:
   ```bash
   mvn spring-boot:run
   ```
3. Access the endpoint:
   Open your browser and go to [http://localhost:8080/hello](http://localhost:8080/hello)

You should see:
```
Hello, World!
```
